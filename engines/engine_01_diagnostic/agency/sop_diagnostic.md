## Tujuan

## Checklist SOP

### Data Intake
- [ ] Dataset diterima lengkap
- [ ] Format file valid
- [ ] Nama klien & proyek dicatat

### Diagnostic
- [ ] Engine dijalankan tanpa error
- [ ] Semua modul aktif
- [ ] Laporan utama ter-generate

### Review
- [ ] Skor kualitas masuk akal
- [ ] Risiko tertulis jelas
- [ ] Bahasa non-teknis

### Delivery
- [ ] File dikirim ke klien
- [ ] Catatan rekomendasi disertakan
- [ ] Arsip laporan disimpan

## Catatan Penting
Engine ini bersifat read-only.
Tidak boleh mengubah data klien.
