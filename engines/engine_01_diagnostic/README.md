# Raw Data Diagnostic Engine

Engine profesional untuk menganalisis kondisi data mentah
sebelum digunakan untuk analisis, automation, atau keputusan bisnis.

## Filosofi
Saya tidak menjual pembersihan data.
Saya menjual kejelasan sebelum keputusan dibuat.

## Apa yang Dilakukan Engine Ini
- Mendeteksi risiko data (missing, duplikat, schema, encoding)
- Memberikan laporan objektif kondisi dataset
- Membantu keputusan: lanjut, perbaiki, atau tolak data

## Apa yang TIDAK Dilakukan
- Tidak membersihkan data
- Tidak mengubah isi dataset
- Tidak menggantikan business judgment
