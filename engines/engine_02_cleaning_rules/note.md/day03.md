## Day 3 – Buat Kerangka Folder Engine
**Tujuan:** Struktur project modular, profesional, siap GitHub.  
**Langkah:**
- Buat root folder `data_cleaning_rule_engine/`

**Fokus:** Standarisasi project → maintainable, modular, siap di-GitHub.
